# Cafe Sales Total Spent Prediction ☕

Machine learning project to predict customer total spending using cafe sales data.

## Overview
The project includes data cleaning, EDA, feature encoding, and model training using:

- Linear Regression  
- KNN  
- XGBoost  

Models are evaluated with MSE and R² to find the best performer.

## Tech Stack
Python, Pandas, NumPy, Scikit-learn, XGBoost, Jupyter Notebook

## How to Run
1. Install dependencies  
2. Open the notebook  
3. Run `Total_Spent_Prediction_For_a_Cafe_Sales.ipynb`
